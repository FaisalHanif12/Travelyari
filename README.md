### Run Locally

To run **Tourly** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/tourly.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/tourly.git
```
